# シニアおしゃべりAI

高齢者向けの音声チャットアプリです。話しかけるだけでAIと会話ができます。

## 🎯 特徴

- **ワンタッチ起動**: アプリを開いたらすぐに使える
- **音声ファースト**: 文字入力は一切不要
- **安心の可視化**: 話した内容が文字で表示される
- **記憶の蓄積**: 過去の会話を読み返せる

## 🛠️ セットアップ

### 1. OpenAI APIキーの取得

1. [OpenAI Platform](https://platform.openai.com/) にアクセス
2. アカウントを作成またはログイン
3. API Keys セクションで新しいキーを作成
4. `sk-` で始まるキーをコピー
5. **初回登録時は$5の無料クレジットが付与されます**（約1年分の利用可能）

### 2. ローカルで実行

```bash
# このディレクトリに移動
cd senior-voice-chat

# ローカルサーバーを起動（Pythonの場合）
python -m http.server 8000

# または Node.js の場合
npx serve
```

ブラウザで `http://localhost:8000` にアクセス

### 3. APIキーの設定

1. アプリの「⚙️ 設定」ボタンをタップ
2. APIキー入力欄にコピーしたキーを貼り付け
3. 「保存する」ボタンをタップ

## 📱 使い方

### 基本の会話

1. 「🎤 話しかける」ボタンをタップ
2. AIの挨拶を聞く
3. 「タップして話す」ボタンを押して話しかける
4. AIの返答を聞く
5. 会話を続ける

### 話題を選ぶ

ホーム画面で話題を選んでから会話を始められます：
- ☀️ 今日の天気
- 🏥 健康について
- 👨‍👩‍👧‍👦 家族の話
- 📺 趣味の話

### 過去の会話

「📁 過去の会話」ボタンから、過去の会話履歴を見ることができます。

## 🔒 プライバシー

- 会話データは**あなたの端末内のみ**に保存されます
- サーバーには会話内容を送信しません（AIへの問い合わせを除く）
- いつでも履歴を削除できます

## ⚙️ 設定項目

### 音声速度
- ゆっくり: 0.7倍速
- 普通: 0.9倍速（デフォルト）
- 速い: 1.1倍速

### 文字サイズ
- 普通: 16px
- 大きい: 18px（デフォルト）
- 特大: 22px

## 📋 動作環境

- **推奨ブラウザ**: 
  - iPhone: Safari (iOS 15以降)
  - Android: Chrome
  - PC: Chrome, Edge

- **必要な権限**:
  - マイク（音声入力用）
  - インターネット接続（AI通信用）

## 🔧 トラブルシューティング

### マイクが動作しない

1. ブラウザの設定でマイクを許可
2. iPhone: 設定 > Safari > マイク > 許可
3. ページを再読み込み

### 音声が聞こえない

1. 端末の音量を確認
2. サイレントモードを解除
3. ブラウザを再起動

### AI応答が遅い

1. インターネット接続を確認
2. WiFiに接続
3. しばらく待ってから再試行
