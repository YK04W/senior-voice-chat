## 📋 次のステップ: GitHub Pagesを有効化

### 手順

1. **GitHubリポジトリを開く**
   ```
   https://github.com/YK04W/senior-voice-chat
   ```

2. **Settings をクリック**
   - リポジトリページの上部メニューから「Settings」を選択

3. **Pages をクリック**
   - 左サイドバーの「Pages」をクリック

4. **Source を設定**
   - **Source**: `Deploy from a branch` を選択
   - **Branch**: `main` を選択
   - **Folder**: `/ (root)` を選択
   - **Save** をクリック

5. **デプロイ完了を待つ**
   - 2-5分程度かかります
   - 「Your site is live at」というメッセージが表示されます

## 🌐 アクセスURL

デプロイ完了後、以下のURLでアクセス可能になります：

```
https://yk04w.github.io/senior-voice-chat/
```

## 📱 iPhoneでのアクセス

1. **Safariで開く**
   - 上記URLをSafariで開く

2. **ホーム画面に追加**
   - 共有ボタン（□↑）をタップ
   - 「ホーム画面に追加」を選択
   - 名前を「AIおしゃべり」に設定
   - 「追加」をタップ

3. **APIキーを設定**
   - アプリを開く
   - 「⚙️ 設定」をタップ
   - OpenAI APIキーを入力
   - 「保存する」をタップ

4. **動作確認**
   - 「🎤 話しかける」ボタンをタップ
   - マイク権限を許可
   - 会話を開始

## 🔧 トラブルシューティング

### 404エラーが表示される場合

- デプロイに数分かかることがあります
- 5-10分待ってから再度アクセス
- GitHubリポジトリの「Actions」タブでデプロイ状況を確認

### ページが更新されない場合

- ブラウザのキャッシュをクリア
- シークレットモードで開く
- `Ctrl + F5`（Windows）または `Cmd + Shift + R`（Mac）で強制リロード
