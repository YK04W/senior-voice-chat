# 音声入力の改善

## 修正内容

### 1. 録音という概念を削除
- 「録音中」という表現を削除
- 「聞いています...」に変更
- シニアユーザーが構えないように配慮

### 2. ボタンの色を緑に変更
- 赤色（録音）→ 緑色（聞いています）
- より穏やかな印象
- アニメーションも緑色に

### 3. 音声認識の改善
- `continuous: false` に戻す（一発で認識）
- デバッグログを追加
- エラーハンドリングを改善

### 4. AI挨拶の改善
- 挨拶も会話履歴に保存
- 会話の文脈を保持

## 使い方

1. 「タップして話す」ボタンをタップ
2. 「聞いています...」と表示される（緑色）
3. 話す
4. 話し終わると自動的に文字起こし
5. AI応答が返る

## デバッグ方法

ブラウザのコンソール（F12）で以下を確認：
- 「音声入力開始」
- 「聞き取り中: ...」
- 「音声認識結果: ...」
- 「AI応答を取得開始...」
- 「APIレスポンス受信: 200 OK」

## 次のステップ

もし会話がまだ成立しない場合は、以下を確認してください：
1. APIキーが正しく保存されているか
2. ブラウザのコンソールにエラーが表示されていないか
3. マイクの権限が許可されているか
