# 🚀 iPhoneデモ - 即座にアクセスする方法

## ⚡ 方法1: ローカルネットワーク（今すぐ試せる）

### 手順

1. **PCとiPhoneを同じWi-Fiに接続**
   - 必ず同じWi-Fiネットワークに接続してください

2. **PCでサーバーを起動**
   ```bash
   cd senior-voice-chat
   npx serve -p 3000 --cors
   ```
   （既に起動している場合はスキップ）

3. **PCのIPアドレスを確認**
   - 現在のIPアドレス: **192.168.1.82**

4. **iPhoneのSafariで開く**
   ```
   http://192.168.1.82:3000
   ```

5. **ホーム画面に追加**
   - Safariの共有ボタン（□↑）をタップ
   - 「ホーム画面に追加」を選択
   - 名前を「AIおしゃべり」に設定

### ⚠️ 注意事項

- **ファイアウォール設定**: Windowsファイアウォールでポート3000を許可する必要がある場合があります
- **HTTPSが必要**: 音声認識はHTTPS接続が必要な場合があります（ローカルネットワークではHTTPでも動作する場合あり）

## 🌐 方法2: GitHub Pages（推奨・デモ当日用）

### 今すぐデプロイする手順

1. **GitHubでリポジトリを作成**
   - [github.com](https://github.com) にログイン
   - 「New repository」をクリック
   - リポジトリ名: `senior-voice-chat`
   - Public を選択
   - 「Create repository」をクリック

2. **コードをプッシュ**
   ```bash
   cd senior-voice-chat
   git remote add origin https://github.com/YOUR_USERNAME/senior-voice-chat.git
   git branch -M main
   git push -u origin main
   ```
   （YOUR_USERNAMEを実際のGitHubユーザー名に置き換えてください）

3. **GitHub Pagesを有効化**
   - リポジトリの **Settings** → **Pages**
   - **Source**: `Deploy from a branch` を選択
   - **Branch**: `main` / `/ (root)` を選択
   - **Save** をクリック

4. **数分待つ**
   - デプロイには2-5分かかります
   - デプロイ完了後、以下のURLでアクセス可能：
   ```
   https://YOUR_USERNAME.github.io/senior-voice-chat/
   ```

5. **iPhoneでアクセス**
   - Safariで上記URLを開く
   - ホーム画面に追加

### ✅ メリット

- HTTPS対応（音声認識に最適）
- どこからでもアクセス可能
- URLが固定で共有しやすい
- デモ当日に安心

## 🔧 トラブルシューティング

### 問題: iPhoneからアクセスできない（ローカルネットワーク）

**解決策**:
1. PCとiPhoneが同じWi-Fiに接続されているか確認
2. Windowsファイアウォールでポート3000を許可：
   - Windowsセキュリティ → ファイアウォールとネットワーク保護
   - 詳細設定 → 受信の規則 → 新しい規則
   - ポート → TCP → 3000 → 許可

### 問題: マイクが動作しない

**解決策**:
1. iPhone設定 → Safari → マイク → 許可
2. ページを再読み込み
3. Safariでアクセスしているか確認（Chrome等では動作しない場合あり）

### 問題: APIキーエラー

**解決策**:
1. APIキーが正しく入力されているか確認（sk-で始まる）
2. OpenAI PlatformでAPIキーが有効か確認
3. クレジット残高を確認（初回$5クレジット）

### 問題: 音声認識が動作しない

**解決策**:
1. **HTTPS接続を使用**（GitHub Pages推奨）
2. Safariでアクセス（Chrome等ではWeb Speech APIが動作しない場合あり）
3. マイク権限が許可されているか確認
